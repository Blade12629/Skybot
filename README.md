# Skybot
osu! tourney bot

---
Currently Features:

- Analyzer
- Statistics for user and teams
- Verification
- API (WIP)

- And much more to come!

- .Net Core 3.1
- Discord Bot Account
- Osu Account
- MariaDB/MySQL Database
